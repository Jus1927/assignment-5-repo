How to Apply CSS to HTML

As explained in mdn web docs, we can reference an external CSS file using a <link> element. Place this <link> element in the <head> element of our HTML file. The href attribute needs to contain the file in our file system that contains the CSS stylesheet. As an example, it might look like this:

<link rel="stylesheet" href="myStyles.css"/>.

CSS will then be applied based on the selectors and rules defined in the CSS file based on their importance and specificity.

How to Use JavaScript in a Webpage

It is possible to include JavaScript in an HTML file either by linking an external JavaScript file or by including JavaScript code directly within the HTML file. This mdn web docs page explains how to include JavaScript using both of the methods describe above.

Linking an External JavaScript File:

In order to execute a JavaScript script, you can include a <script> tag. The src attribute will contain a file path to the JavaScript file you want to execute. An example <script> tag is shown below:

<script src="scripts/someScript.js"></script>

Writing JavaScript with an HTML file:

Alternatively, you can include JavaScript code directly within the HTML file by putting the code between <script> tags. An example of this is shown below:

<script>

console.log("This is a JavaScript statement");

</script>

If you inspect the page, you can see this statement printed in the console.
